# nano-url

A URL shortener built with Hono, React, and PostgreSQL.

## Getting Started

### Prerequisites

- Node.js >= 20
- pnpm

### Install

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Docker

```bash
docker compose up -d
```

## Environment Variables

Copy `.env.example` to `.env` and fill in the values:

```bash
cp .env.example .env
```
